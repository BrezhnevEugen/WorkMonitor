# Changelog

Формат: [Keep a Changelog](https://keepachangelog.com/ru/1.1.0/). Версионирование: [SemVer](https://semver.org/lang/ru/).

## [2.0.0] — 2026-04-18

Полный редизайн интерфейса в тёмном техническом стиле. Одна большая визуальная и UX-итерация; обратная совместимость с пользовательскими настройками v1 не ломается — старые данные просто не читаются.

### Добавлено

- **Полоса метрик вверху panel'а.** CPU, RAM, Disk (свободно), Network (скорость скачивания) — по мини-бару на каждую, цвет реагирует на пороги (зелёный / жёлтый / красный).
- **Поддержка тем.** Четыре варианта: `Авто (по системе)` — следит за macOS dark/light и переключается на лету; `Тёмная терминальная` (зелёный акцент, шрифт JetBrains-style моно); `Светлая минималистичная`; `Полночь` (синяя тёмная). Выбор через radio-picker со свотчами палитры. Подписка на `AppleInterfaceThemeChangedNotification` — Auto-режим реагирует мгновенно на смену системной темы.
- **Экран настроек.** Шестерёнка в шапке popover'а. Блоки: тема, отображаемые разделы, группировка портов, язык, автозапуск.
- **Видимость разделов.** Любой из Ports / Docker / Processes можно скрыть (чекбоксы, сохраняются в `UserDefaults`).
- **Группировка портов по приложению.** Включается в настройках (по умолчанию включена); один процесс с несколькими слушающими портами отображается одной свёрнутой группой.
- **Hover-действия на строках.** Наведение на порт — `open` / `kill`; на контейнер — `stop/start` / `logs` (UI-заглушки для команд, будут в 2.1).
- **Live-индикатор в футере.** Пульсирующая точка + надпись `live · 2s`.
- **Компактный About.** Краткое описание + версия + кнопка поддержки.

### Изменено

- **Иконка в меню-баре полностью монохромная** (template-изображение SF Symbol). macOS автоматически красит её в белый на тёмной панели и чёрный на светлой. Состояние теперь кодируется формой символа: заполненный кружок, пустой кружок, треугольник-предупреждение, круг с восклицательным знаком — а не цветом. Рядом — счётчик открытых портов в системном `labelColor`.
- **Размеры popover'а.** 480×560 → 420×600. Плотнее информация на единицу площади.
- **Частота обновления.** 5 с → 2 с при открытом окне, плюс фоновый опрос каждые 15 с для актуального счётчика в меню-баре.
- **Версия приложения.** `CFBundleShortVersionString` 1.0.1 → 2.0.0; `CFBundleVersion` 2 → 3.

### Исправлено

- Живая перерисовка при смене темы: при переключении обновляется всё окно сразу, а не только секция, где произошёл клик (заменили антипаттерн `@ObservedObject + Singleton.shared` на корректный `@EnvironmentObject`).

### Под капотом

- Новый `WorkMonitorCore` `TopCPUOutputParser`, `DfOutputParser`, `NetstatOutputParser` + юнит-тесты на них.
- Новый модуль `Theme.swift`: `ThemePalette`, `ThemeKind`, `ThemeManager`.
- Новый модуль `AppSettings.swift` — постоянные пользовательские флаги через `UserDefaults`.
- `SystemMonitor` расширен полями `@Published cpu/disk/network` и методами `fetchCPU/fetchDisk/fetchNetTotals`; скорость сети считается дельтой между сэмплами.
- Локализация: +20 ключей в `ru/en.lproj/Localizable.strings` (metric_*, section_*, settings_*, theme_*, updated_*).

## [1.0.0] — 2026-04-17

Первый публичный релиз.
