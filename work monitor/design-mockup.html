Tray icon · состояния

Иконка в menubar меняется в зависимости от состояния dev-окружения. Маленький текстовый индикатор (wm · N) показывает количество активных dev-портов — понятно с одного взгляда.

wm · 8 wm · 0 wm · 12 wm · RAM

ok — всё работает · idle — ничего не запущено · warn — unhealthy контейнер или много портов · crit — память > 90% или упал docker.

Информационная структура

Header env + host + timestamp Summary CPU · RAM · Disk · Net (всегда видно) Ports кто слушает что, процесс, pid, kill/open Docker контейнеры со статусом, портами, памятью Processes топ по RAM/CPU — где утечка Services brew services, launchd, nginx Footer live-индикатор, settings, quit

Поведение

• Клик по иконке — popover под ней, как у Stats.
• ⌥+клик — форсированный refresh.
• Правый клик — мини-меню (pause, settings, quit).
• Секции сворачиваются и запоминают состояние.
• Hover по строке — появляются действия (kill / logs / open).
• Сетевые данные обновляются каждые 2с, порты/docker — каждые 5с.
